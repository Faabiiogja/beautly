host === "localhost" / preview → tenant de dev do .env

MRR no Super Admin — como calcular

-- simples e direto para MVP
SELECT COUNT(*) as total_tenants,
       COUNT(*) * :monthly_price as mrr
FROM tenants
WHERE status = 'active';
O preço mensal vive no .env no MVP. Quando houver planos, migra para tabela.
