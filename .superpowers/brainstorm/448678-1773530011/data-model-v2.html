Fluxo de criação de tenant

1. INSERT INTO tenants → id
2. INSERT INTO professionals (name = nome do tenant, is_active = true) → default professional
3. INSERT INTO business_hours (seg-sex, 09:00–18:00) → grade padrão
4. Supabase Auth: create user → INSERT INTO tenant_users
5. Tenant pronto para usar
